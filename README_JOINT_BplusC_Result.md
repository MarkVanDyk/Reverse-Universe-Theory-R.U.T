## JOINT(B+C) headline result

The current strongest controlled result in this repository comes from the **near-frozen dynamic-ω, bS-priority full refit** in the JOINT(B+C) pipeline.

### Headline comparison
- **ΛCDM**: 1691.76
- **w0waCDM**: 1693.21
- **Frozen-ω RUT**: 1717.43
- **Dynamic-ω, bS-priority near-frozen full refit**: 1513.09

### Main interpretation
The numerical pattern now suggests that:

- a perfectly frozen ω is too rigid
- a fully unconstrained dynamic ω is too loose
- a **slightly evolving, near-frozen ω** gives the best controlled result in the present setup
- the strongest controlled improvement appears when the **sine-channel freedom** is allowed more movement than the cosine-channel freedom

### Robustness
A local robustness test around the best-fit point repeatedly converged back to essentially the same basin:

- reference BIC: 1513.0878
- best local BIC: 1513.0391
- mean local BIC: 1513.0979
- standard deviation: 0.1407

### Important caution
This is the strongest controlled numerical result so far, but it is **not yet final proof**. Additional work still in progress includes:

- boundary-sensitivity checks
- stronger growth/RSD modeling
- fresh-environment replication
- further benchmark comparisons
